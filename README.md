# Math Puzzle

A geometric puzzle showing a visual math problem with colored triangles. The puzzle asks "What is the blue area?" based on the given triangle areas.

## Features

- Clean, educational geometric visualization
- Color-coded triangular sections with area labels
- Interactive SVG-based diagram
- Modern, responsive design

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Build for production: `npm run build`

## Technologies

- React
- TypeScript
- Tailwind CSS
- Vite
- SVG graphics

## Problem Description

The puzzle presents a square divided into colored triangular sections:
- Red triangle: area = 2
- Yellow triangle: area = 3  
- Green triangle: area = 4
- Blue triangle: area = ?

The goal is to determine the area of the blue section based on the geometric relationships.
